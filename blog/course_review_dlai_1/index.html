Course Review - Intro To Tensorflow

April 14, 2019    review

Introduction to TensorFlow for AI, ML, & DL

(Source: deeplearning.ai)

First and foremost, thanks to Coursera and deepLearning.ai for granting me the scholarship to take this course. I owe them. :)

I have planned to write an honest and short review about the course, I just completed it today. Things that you need to know before taking this course, whether it is suitable for you and so on. So, let us directly jump in.

Expects Prerequisite Knowledge

Professor Laurence Moroney and Andrew NG are the backbones behind this course (picture below). They know that they already have great foundational courses in this domain, and that is probably one of the reasons this course expects you to have intermediate knowledge in machine learning and deep learning in general. This is not the course to build you fundamentals, rather it is to get you started with Tensorflow, to be apt.

(Source: deeplearning.ai)

Great Community Support

Coursera is famous for its forums, and it actually helps you out if you are facing some doubt regarding the lessons. I got direct replies from Prof.Moroney who is a developer advocate at Google working on AI with Tensorflow.

Byte Sized Lessons

This is something that I really look for, I prefer short lessons more than long 30-40 minute talks. This course takes into consideration that factor.

Really Cool Projects

I have done several projects in this domain, but to me atleast 40% of the coursework projects were new. All the projects were very useful and actually carry weightage in the industry. Even if you have done the projects, you will learn something new. I am not naming the projects here, due to copyright issues, but you can check out my GitHub or Projects tab for more details.

(Source: hanxiao.github.io)

Code Centric Teaching

Prof.Moroney has been awesome in explaining every single module and line of code that he writes. He has explained the neural network models with utmost care, all you need to know is some Python syntax and basic Machine Learning and DNN. (Source: deeplearning.ai)

Absolutely Interesting Modules

There were four modules that were mainly discussed in the course. The first one was mostly introductory, but the remaining were very interesting and dominant topics that needs to be studied. One of the main areas of concern in the course has been Convolutional Neural Networks for image based classification. More details regarding this can be found here. (Source: adeshpande3.github.io)

Very Fast Paced & Shallow

The course did not go much deep into the nitty gritties of the working of Tensorflow. So basically, you will be learning how to use a framework. You will not be able to understand what happens beneath the abstraction layer. This point has been criticized by some students who expected a bit more from the course. At times, I realised as if I am completing the course too fast, or probably a bit more weeks of material could have been easily added.

Personal Reviews

This course is good to get you started with Tensorflow. As it seemed to me, they don’t try to make you Tensorflow experts or contributors. Once you have got the start, you have to take it further and learn more about it. I have used PyTorch as well, both of them have their own advantages and disadvantages, so that comparison, for some other day.

Overall, I would highly recommend this course, if you’re looking forward to know more about Tensorflow or if you have not used it yet but satisfy the prerequisites.
